## Instruções de utilização
  * Renomeie **.env.template** para **.env** e configure-o
  * pip install -r requirements.txt
  * flask db init
  * flask db migrate
  * flask db upgrade
  * python wsgi.py (ou utilize servidor wsgi para produção)
